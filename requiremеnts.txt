1. API (Application Programming Interface) — это посредник между
 разработчиком приложений и какой-либо средой,
 с которой это приложение должно взаимодействовать.
 API упрощает создание кода, поскольку предоставляет набор готовых классов,
 функций или структур для работы с имеющимися данными.
 2. Методы представляют собой условные команды, которые соответствуют той
 или иной операции с базой данных — получению информации, записи или удалению. Например:

users.get — метод для получения информации о пользователе.
video.add — метод для добавления видео в список видео.
likes.delete — метод для удаления отметки «Нравится».
